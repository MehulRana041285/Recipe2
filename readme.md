# Recipe App
This is a react native application for those who loves to cook. Following are it's features:
- Categorized Cusines
- Detailed preparation list
- Ingredients list
- Favourite a recipe for offline
- Add ingredients to shopping list


### Libraries used
- Redux
- Redux Thunk
- Realm DB
- React Router Flux
